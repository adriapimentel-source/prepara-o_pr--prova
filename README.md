# 🌿 Descompressão Pré-Prova — Ecos da Margem

Aula interativa sobre cuidado emocional e técnicas de descompressão para adolescentes e adultos em preparação para provas (PSC e ENEM).

**Professora:** Ádria Pimentel Silva  
**Especialidade:** Educadora Popular e Psicóloga (CRP 20/12.882)  
**Instituição:** Ecos da Margem — Rede Emancipa Manaus

---

## 📋 Conteúdo da Aula

A aula é composta por **12 slides interativos** com duração total de **50 minutos**:

1. **Capa** — Apresentação e contexto
2. **Chegada e Contexto** — Espaço seguro e propósito
3. **Os Três Quadros do Rio** — Reflexão sobre estado emocional
4. **Mapeamento dos Desafios** — Identificação de sintomas comuns
5. **Estratégias para a Véspera** — Baseadas em evidências científicas
6. **Relaxamento Muscular Progressivo** — Técnica de Jacobson
7. **Dia da Prova** — Visão geral das 5 técnicas
8. **Respiração em Caixa** — Box Breathing (4-4-4-4)
9. **Grounding 5-4-3-2-1** — Ancoragem sensorial
10. **Dump Cognitivo** — Descarregar pensamentos intrusivos
11. **Ressignificar a Ansiedade** — Pesquisa de Alison Wood Brooks
12. **Fechamento** — Mensagem de encerramento

## 🚀 Como Usar

### Localmente
1. Baixe o arquivo `index.html`
2. Abra em qualquer navegador (Chrome, Firefox, Safari, Edge)
3. Use as setas do teclado (← →) ou clique nos botões para navegar

### No GitHub Pages
1. Crie um repositório no GitHub chamado `ecos-aula-interativa`
2. Faça upload do arquivo `index.html`
3. Vá em **Settings** → **Pages** → Escolha **Deploy from a branch** → Selecione **main** e **/root**
4. Seu site estará em: `https://seu-usuario.github.io/ecos-aula-interativa`
